# KompothLib

A simple API to extract bibliographical data from multiple sources.

Created for my personal needs.

## Usage

In development:
```bash
uvicorn app.main:app --reload
```

## Sources

- [x] FantLab API
- [x] Google Books

## Expected output

A list of books with following data:

- Title
- Year of publication
- Authors
- Poster image link
- Description
